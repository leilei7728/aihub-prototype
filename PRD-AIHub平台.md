## 一、产品概述

### 1.1 产品定位

AIHub 是一款面向个人开发者和 AI 重度用户的**本地 AI 工具管理平台**。用户通过 AIHub 一站式完成主流 AI 客户端的安装管理、模型路由配置与 Tokens 消耗管理，无需逐一配置各工具的 API Key 和接入地址。

### 1.2 核心价值

- **统一入口**：一个平台管理所有已安装的 AI 工具，无需在多个客户端之间切换配置
- **模型路由**：本地网关（`127.0.0.1:15721`）拦截所有 AI 请求，由 CC Switch 完成协议适配和模型分发
- **Tokens 经济**：用户购买平台 Tokens，平台统一调用各模型 API，用户无需自购多个厂商 Key
- **智能调度**：根据任务难度、响应速度、额度状态自动选择最合适的模型

### 1.3 目标用户

| 用户类型 | 描述 |
|----------|------|
| AI 开发者 | 重度使用 Claude Code、Codex 等编程工具，需要多模型灵活切换 |
| 效率用户 | 日常使用 Claude Desktop、OpenCode 等对话工具，追求稳定和省心 |
| 企业小团队 | 统一管理团队成员的 AI 工具用量，集中采购 Tokens |

## 二、整体架构

### 2.1 系统架构

```
用户 AI 工具（Claude / Codex / OpenCode 等）
        ↓ API 请求
本地网关 127.0.0.1:15721
        ↓ 协议适配 & 模型路由
CC Switch（本地服务）
        ↓ 按路由策略转发
各模型 API（Anthropic / OpenAI / Google / DeepSeek 等）
```

### 2.2 页面结构

```
AIHub
├── P01 总控台
├── P02 AI 软件
├── P03 模型路由
├── P04 Tokens 中心
├── P05 购买（含订单记录 Tab）
└── P07 设置
```

## 三、功能模块详细说明

### 3.1 登录 / 注册

**交互流程：**
1. 用户输入手机号
2. 点击"获取验证码"，按钮进入 60 秒倒计时
3. 输入验证码后点击"登录 / 注册"，未注册手机号自动创建账号
4. 登录成功后可选设置密码（下次可免验证码登录）

**规则：**
- 未登录可浏览总控台，点击其他页面弹出登录引导
- 需要登录的功能：AI 软件安装、模型路由配置、Tokens 管理、购买、设置

#### 3.4.4 新增模型（自定义接入）

用户可接入任意兼容 OpenAI 格式的外部模型：

**配置项：**
- 模型展示名称（仅用于展示，不影响实际调用）

- API Key

## 五、数据说明

### 5.1 Tokens 计算规则

- 只统计通过本平台本地网关实际转发的请求消耗
- 用户自行调用 API（绕过网关）的消耗不计入
- 一条消息可能产生多次请求（重试、补全上下文、工具调用）

### 5.2 套餐消耗优先级

- 多套餐并存时，按到期时间最近的优先消耗（先到期先用）

### 5.3 余额预警

- 剩余 Tokens < 20,000 时触发系统通知（可在设置中关闭）
